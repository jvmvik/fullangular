Fresh Java/JS web framework playing reactive modern technologies !

Built with the top of the latest technologies..
The core is animated by Java 8+ supporting Nashorn JavaScript engine.
The backend and frontend share the same language and pattern both are written in JavaScript
Netty 4 power a simple efficient reactive back-end made of fluent API, and the best of java world
The web application can run real-time Websocket layer..
Ligth weight application runs standalone on low power chip like Raspbery PI..

A collection of example illustrates the integration between all these technologies: Netty, Milkyway, AngularJS, BootStrap 3.0...
